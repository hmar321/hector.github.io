MongoDB Atlas

Héctor Almaraz-Carlos Escalera-Alberto Sánchez

MongoDB Atlas en Google Cloud. Una base de datos mundial basada en la nube y completamente administrada de MongoDB que combina modelos de datos similares a JSON, indexación y búsqueda avanzadas, y escalabilidad elástica, a la vez que automatiza las tareas administrativas que llevan mucho tiempo.

1-Descargarse la extensión de MongoDB for VS code:

2-En la pestaña de MongoDB dirígete a Mongo Atlas desde la pestaña de conexiones. Para abrir la pestaña puedes presionar Add Connection o desde la consola(Ctrl+shift+p) escribir Mongo: Connect

3-El link te llevará hasta la página web de mongoDB en la que deberás registrarte para poder crear tu propia base de datos gratuita en la nube.

4-Puedes ingresar desde el método que prefieras, en este ejemplo se hará desde una cuenta google.

5-Selecciona el plan gratuito.

6-Ahora vas a personalizar como te vas a conectar a tu base de datos y donde quieres tu base de datos. Vamos a crear usuarios con contraseña como método de conexión.

7-A continuación selecciona Cloud Environment para crear tu entorno en la nube y selecciona desde que dirección ip se pueden conectar a tu base de datos.

 

8-Para conectarte a tu base de datos en la nube dirígete a Database y selecciona Connect.

9-Copia la línea que te facilita mongo y vuelve a VScode.

10-Pega la línea que has copiado y no te olvides de cambiar <password> por tu contraseña.

11-Por último selecciona Create New Playground que te ofrecerá algunos ejemplos prácticos de operaciones CRUD.
